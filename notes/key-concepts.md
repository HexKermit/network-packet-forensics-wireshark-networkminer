# Key Concepts

## Packet Analysis

Packet analysis means reviewing network traffic to find useful security evidence.

## Wireshark

Wireshark helps analysts inspect packets, apply filters, and follow protocol streams.

## NetworkMiner

NetworkMiner helps extract hosts, images, files, messages, credentials, and other artifacts from packet captures.

## Cleartext Protocols

Protocols like TELNET and FTP can expose usernames, passwords, and commands in plaintext.

## Secure Alternatives

- TELNET → SSH
- FTP → SFTP or SCP
- HTTP → HTTPS
